# Gold Price & CPI Time Series Analysis - VAR (Myanmar)

[![DOI](https://img.shields.io/badge/DOI-10.5281/zenodo.19190323-blue)](https://doi.org/10.5281/zenodo.19190323)

This repository contains the code and analysis for a time series econometric study examining the relationship between 24-carat gold prices and the Consumer Price Index (CPI) in Myanmar, using a Vector Autoregression (VAR) model.

## 📊 Objective
The main objective of this study is to analyze the time series dynamic relationship between gold prices and inflation (CPI) using a Vector Autoregression (VAR) model.

## 🧠 Methodology
- Time Series Analysis
- Vector Autoregression (VAR)
- Stationarity Testing (ADF Test)
- Lag Length Selection
- Granger Causality Test
- Impulse Response Functions (IRF)
- Variance Decomposition

## 📁 Dataset
- 24K Gold Price (Myanmar)
- Consumer Price Index (CPI)
- Frequency: [Monthly]
- Source: [Myanamar Central Statistical Organization]

## ⚙️ Tools & Libraries
- Python / R / Stata / SPSS
- pandas, statsmodels, matplotlib
